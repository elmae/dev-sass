## Requerimientos

se requiere de conocimientos basicos de CSS, HTML ademas de usar node.js

- [Node.js](https://nodejs.org/)
- [Sass](https://sass-lang.com/)

## Installación y como empezar

   - npm init
   - npm install
   - npm run start
